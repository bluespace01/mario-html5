How to build:

Install nvm:
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash

npm 5.x 是与 Node.js 8.x 版本关联的，所以你需要安装 Node.js 8.x（例如 8.17.0），这将自动安装相应的 npm 5.x。

nvm install 8
nvm use 8
npm --version
npm install -g npm@5 

npm install
npm run build


mv https_server.py into the dist;
mkcert localhost;
